# O'zbekiston Buxgalteriya Amaliyoti - O'quv Repozitoriyasi

Ushbu loyiha buxgalteriya sohasida yangi boshlovchilar uchun **Kassa hujjatlarini** to'g'ri to'ldirishni o'rgatish maqsadida yaratilgan.

## Loyiha tarkibi

GitHub repozitoriy quyidagi fayllarni o'z ichiga oladi:

1.  **index.html** - Asosiy menyu sahifasi.
2.  **kko.html** - Kassa Kirim Orderi (KKO-1 shakl) mashg'uloti.
3.  **kcho.html** - Kassa Chiqim Orderi (KChO-2 shakl) mashg'uloti.
4.  **cashbook.html** - Kassa kitobini yuritish mashg'uloti.
5.  **js/data.js** - Buxgalteriya schyotlar rejasi (O'zbekiston standartlari asosida).
6.  **css/styles.css** - Loyiha dizayni.

## Qanday ishlatiladi?

1.  Ushbu repozitoriyni yuklab oling yoki `git clone` qiling.
2.  `index.html` faylini brauzerda oching.
3.  Kerakli bo'limni tanlang (KKO, KChO yoki Kassa Kitobi).
4.  Formalarni to'ldiring:
    *   **KKO va KChO**: Sana, Summa, Schyotlar (Debit/Kredit), Asos va Ilova qismlarini to'ldiring.
    *   **Kassa Kitobi**: Kirim va Chiqim operatsiyalarini kiriting, kunlik qoldiq avtomatik hisoblanadi.
5.  "Chop etish" tugmasi orqali hujjatni qog'ozga chiqarib oling.

## O'qituvchilar uchun

Ushbu qo'llanma talabalarga amaliy mashg'ulot sifatida ishlatilishi mumkin. Talabalar mustaqil ravishda hujjatlarni to'ldirib, natijani PDF shaklida yoki qog'ozda topshirishlari mumkin.
